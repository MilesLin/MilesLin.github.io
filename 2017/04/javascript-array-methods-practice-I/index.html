Array.from()

用這種方式就可以建立一個新的 Array ，這樣 array2 更改後，就不會動到 array1 了

另一種的方法也可以用來複製 Array ，並且也可以將該 Array push 到另一個 Array

arrow functions 用法

使用 Array arrow functions 可以對要複製的 array 做一些操作，使用起來就跟 C# linq 的 Select 差不多。

Array.isArray()

Array.prototype.concat()
