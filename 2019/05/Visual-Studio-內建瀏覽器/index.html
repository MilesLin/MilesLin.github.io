[IDE] Visual Studio 內建瀏覽器

May 08, 2019 in IDE

我通常在開發程式的時候，都喜歡把 Visual Studio 設定成全螢幕模式(Alt+Shift+Enter)，因為很討厭其他視窗的干擾。 而在開發程式的時候，有時候會想要 Google 一下，但是又不很想離開 Visual Studio 視窗，還好 Visual Studio 有提供內建瀏覽器。

打開內建瀏覽器的方式

瀏覽器可以從 View > Other Windows > Web Browser 開啟，或者用快捷鍵 Ctrl+ALt+R 開啟。 開啟的樣子如下圖。

visual studio built-in browser

修改預設網址

開啟瀏覽器預設網址並不是 Google 首頁，這時候要打開 Tools > Options 底下的 Environment > Web Browser 修改，如下圖。

change-default-url

小結

我平常很喜歡用這個功能來快速 Google ，開發起來會比另外開 Chrome 順暢。
